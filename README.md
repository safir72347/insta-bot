# insta-bot

## Getting Started

Currently Working:
1. Login
2. Fetch Following
3. Fetch Followers
4. Get Unfollowers

### Prerequisites

```
1. Python
2. Selenium
3. Chrome Webdriver

```

### Installing

1. Install Python
```
https://www.python.org/downloads/
```
2. Install Selenium
```
pip install selenium
```
3. Download Chrome WebDrive
```
* Download from (https://chromedriver.chromium.org/downloads) based on your Chrome browser version
* For Ubuntu: Extract the file and move to /usr/local/bin
* For Windows:
```

### Working

Step 1 :
```
Open config.py and add username and password on line 3 and 4
```
Step 2 : 
```
Run bot.py
```
